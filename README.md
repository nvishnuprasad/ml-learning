# ML-Learning

This repository contains notebooks and experiments for learning and practicing Machine Learning concepts.

## Setup

Follow these steps to set up the environment.

### 1. Install Python
Install Python (3.10 or later recommended).

Verify:
```bash
python --version
````

### 2. Install pip

`pip` usually comes with Python. Verify:

```bash
pip --version
```

### 3. Install uv

`uv` is used for fast and reliable dependency management.

```bash
pip install uv
```

Verify:

```bash
uv --version
```

### 4. Clone the repository

```bash
git clone https://github.com/nvishnuprasad/ml-learning/
cd ml-learning
```

### 5. Install dependencies

```bash
uv sync
```

This will install all required dependencies.
